\subsubsection*{template$<$class T$>$\newline
class P\+D\+V$<$ T $>$}

Implementation of {\itshape \mbox{\hyperlink{class_p_d_v}{P\+DV}}} class. 

The {\itshape \mbox{\hyperlink{class_p_d_v}{P\+DV}}} class includes basic attributes like \mbox{\hyperlink{class_p_d_v}{P\+DV}} position, flight time and energy. Some \mbox{\hyperlink{class_p_d_v}{P\+DV}} actions like fly approaching to next target node through G\+PS localization and U\+WB localization, return-\/to-\/home process, inductive power transfer and flight simulation are included as well.

Simulate the whole process. 

The \mbox{\hyperlink{class_p_d_v}{P\+DV}} will depart from the base station and visit the sensor nodes in {\itshape path} one by one, and once it completed all targets in the recharging list, it will return to the base station. Everytime before the \mbox{\hyperlink{class_p_d_v}{P\+DV}} visiting the next target sensor node, it will check its remain energy, if not enough for next visit, power transfer and R\+TH implementation, it execute R\+TH immediately.
