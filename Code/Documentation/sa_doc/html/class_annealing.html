template<class T>
class Annealing< T >

Implementation of Annealing class.

The Genetic class includes basic attributes like target, trail vectors, metric vectors, etc. and actions like reduce temperature, calculate fitness function etc.

Calculate the fitness of the possible solution.

The criteria is based on the amount of recharged energy and the distance of flight. Related formula: Fitness M = alpha * tanh(E_{wsn}) + beta * inv_tanh(d_{pdv}) + gamma * inv_tanh(E_{pdv})

Parameters
sn_listA vector of all sensor nodes.
clst_listA vector of clusters to be calculated (can be one target or trail vector).
Template Parameters

Returns
